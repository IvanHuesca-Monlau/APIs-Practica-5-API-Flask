# Monlau | APIs-Practica-5-API-Flask

¡Bienvenido/a a mi actividad de APIs-Practica-5-API-Flask!


## Instalación

1 - Clona el repositorio de GitHub en tu máquina local:
```bash
git clone https://github.com/IvanHuesca-Monlau/APIs-Practica-5-API-Flask
```

2 - Crea y activa un entorno virtual (recomendado):
```bash
python -m venv venv
.\venv\Scripts\activate
```

3 - Instala las dependencias necesarias para el proyecto:
```bash
pip install -r requirements.txt
```
## Uso del proyecto

1 - Para iniciar el proyecto en modo desarrollo, usa el siguiente comando. Esto arrancará el servidor de Flask localmente:
```bash
python app.py
```

## Tecnologías usadas

- __Python__: Lenguaje de programación principal empleado.
- __Flask__: Microframework de Python utilizado para construir la API y manejar las rutas.
- __SQL__: Para la estructuración y gestión de la base de datos.
